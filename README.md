# Code Flash App

Aplikasi **flashcard** untuk belajar cepat, dibuat menggunakan **React Native** dan **Expo Router**.

---

## Fitur Utama
- **Splash Screen**: menampilkan logo & nama aplikasi
- **Dashboard**: menu untuk
  - Buat Flashcard
  - Perpustakaan
- **CRUD Flashcard**: buat, lihat, edit, hapus flashcard
- **Animasi Flip 3D**: kartu bisa dibalik dengan animasi
- **Drag & Drop**: atur urutan kartu flashcard

---

## Instalasi

1. Clone repository:
```bash
git clone https://github.com/username/code-flash-app.git
```
Masuk folder project:

```bash
cd code-flash-app
```
Install dependencies:
```bash
npm install
```
Jalankan aplikasi:
```bash
npx expo start
```
## Lalu buka di

- iOS Simulator

## Struktur Folder
- app/         → screen & navigasi
- components/  → komponen reusable
- context/     → state global flashcard
- assets/      → logo, gambar, icon
- package.json → konfigurasi project
